# Ontario College of Family Physicians

This project is a complete React + Vite website redesign concept for the Ontario College of Family Physicians, focused on member resources, advocacy, continuing professional development, practice supports, leadership content, events, and physician engagement.

The site is designed as a modern, responsive public-facing experience for Ontario family physicians, with clear content pathways, polished visual sections, accessible layout patterns, and a structure that supports future expansion into a full CMS-driven production website.

The application is structured using reusable React components and route-based pages for maintainability and scalability, with deployment configured for GitHub Pages.

👉 **[View Live Site](https://kyla-zeit.github.io/ontario-college-of-family-physicians/)**

---

## 🚀 Features
- Fully responsive layout (mobile, tablet, desktop)
- Modern nonprofit / healthcare association-style design
- Member-focused navigation for resources, advocacy, events, news, leadership, and practice supports
- Reusable React components and shared layout structure
- Route-based page organization using TanStack Router
- Accessible design direction with readable spacing, contrast, and content hierarchy
- GitHub Pages deployment
- Structured content flow for easier future edits and production expansion
