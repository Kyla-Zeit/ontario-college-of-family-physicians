# OCFP Website Redesign

Live site:

https://kyla-zeit.github.io/ontario-college-of-family-physicians/

Open this folder in VS Code, then run:

```bash
npm install
npm run dev
```

Vite will print a local URL, usually `http://localhost:5173/`.

Useful commands:

```bash
npm run build
npm run build:pages
npm run preview
```

The corrected OCFP content is in `src/routes` and shared layout pieces are in `src/components`.
